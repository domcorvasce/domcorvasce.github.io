# domcorvasce.github.io

## Requirements

- Hugo v0.121.2 or higher
- Git

## Installation

Clone this repository:

```shell
git clone git@github.com:domcorvasce/domcorvasce.github.io
```

Pull the blog theme:

```shell
cd domcorvasce.github.io
git submodule update --init
```

## Getting Started

To run the local server:

```shell
hugo serve
```
